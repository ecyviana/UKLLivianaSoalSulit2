1. Program meminta pengguna memasukkan jumlah baris dan kolom matriks.
2. Program membuat tiga matriks:
- matriksA untuk menyimpan elemen matriks pertama
- matriksB untuk menyimpan elemen matriks kedua
- hasil untuk menyimpan hasil penjumlahannya
3. Pengguna memasukkan elemen-elemen Matriks A satu per satu.
4. Pengguna memasukkan elemen-elemen Matriks B satu per satu.
